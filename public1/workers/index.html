DEMO 6 WEB WORKER HTML5 JAVASCRIPT


- Démarrer Worker : Démarrer ou redémarre le Worker après qu'il ait été suspendu.
- Suspendre Worker : Mettre le Worker en pause, l'instance du Worker est toujours présente. Il peut être redémarré via le bouton Démarrer Worker.
- Stopper Worker : Stopper le Worker, l'instance du Worker est détruite. Il ne peut être redémarré via le bouton Démarrer Worker.
